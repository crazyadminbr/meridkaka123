# 🩸 Кровавый Меридиан — Telegram Bot

Бот для приёма резюме и управления командой проекта **ZEUS COMPANY**.

---

## 🚀 Установка и запуск

### 1. Установить зависимости
```bash
npm install
```

### 2. Настроить .env
```bash
cp .env.example .env
```

Откройте `.env` и заполните:
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id,id_второго_админа
```

> Узнать свой Telegram ID можно у бота [@userinfobot](https://t.me/userinfobot)

### 3. Запустить
```bash
node index.js
```

---

## 📋 Команды бота

| Команда | Кто может | Описание |
|---------|-----------|----------|
| `/start` | Все | Запуск / перезапуск бота |
| `/admin` | Администраторы | Открыть панель администратора |
| `/setrole <id> <role>` | Директор / Ген.директор | Назначить роль пользователю |

## 🗃️ База данных

SQLite файл создаётся автоматически: `meridian.db`

**Таблицы:**
- `users` — зарегистрированные пользователи
- `resumes` — поданные резюме
- `support_tickets` — тикеты поддержки
